# Ecommerce Website - Shopping Cart

A simple ecommerce website built with **HTML**, **CSS**, and **JavaScript**.
The project includes a product listing page and a working shopping cart where users can add products, remove items, update quantities, and view the total price.

## Features

- Responsive ecommerce layout
- Product cards with images, names, and prices
- Shopping cart sidebar
- Add products to cart
- Remove products from cart
- Update product quantity
- Automatic total price calculation
- Clean user interface using Boxicons

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Boxicons
